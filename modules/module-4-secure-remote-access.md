# Module 4: Secure Remote Access

## Overview

Secure remote access allows users to connect safely to internal networks.

---

## VPN (Virtual Private Network)

### SSL VPN
- Browser-based secure connection

### IPsec VPN
- Secure tunnel between networks

---

## Zero Trust Network Access (ZTNA)

- No implicit trust
- Access based on identity and context

---

## Key Concepts

- Encryption of traffic
- Identity verification
- Device compliance checks

---

## Real-world Usage

- Remote employees
- Secure access to internal systems
