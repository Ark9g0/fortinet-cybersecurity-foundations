# Module 3: Authentication and Access Control

## Overview

Authentication verifies identity, while access control defines permissions.

---

## Authentication Methods

- Password-based authentication
- Multi-Factor Authentication (MFA)
- Biometric authentication

---

## Single Sign-On (SSO)

Allows users to log in once and access multiple systems.

---

## Authentication Protocols

- LDAP
- RADIUS
- Kerberos

---

## Access Control Models

- Discretionary Access Control (DAC)
- Mandatory Access Control (MAC)
- Role-Based Access Control (RBAC)

---

## Network Access Control (NAC)

Ensures only compliant devices can access the network.

---

## Best Practices

- Use MFA
- Apply least privilege principle
- Regular access review
