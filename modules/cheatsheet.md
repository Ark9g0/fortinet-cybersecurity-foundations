# Cybersecurity Summary Cheat Sheet

## Cryptography
- Symmetric: same key
- Asymmetric: public/private key
- Hashing: one-way transformation
- PKI: certificate-based trust system

## Network Security
- Firewalls filter traffic
- Zero Trust = never trust, always verify
- Segmentation limits attack spread

## Authentication
- MFA increases security
- SSO simplifies access
- RBAC controls permissions

## Remote Access
- VPN encrypts communication
- ZTNA enforces identity-based access

## Endpoint Security
- EPP prevents threats
- EDR detects attacks
- IoT devices are high risk

## Cloud Security
- Shared responsibility model
- Misconfiguration is a major risk
- API security is critical
