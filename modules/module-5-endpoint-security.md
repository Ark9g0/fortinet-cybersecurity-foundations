# Module 5: Endpoint Security

## Overview

Endpoints are devices like laptops, phones, and IoT devices connected to a network.

---

## Endpoint Security

Protects devices from malware and attacks.

---

## IoT Security

Challenges:
- Weak authentication
- Limited security updates

---

## BYOD

Bring Your Own Device increases risk if not managed properly.

---

## Endpoint Protection (EPP)

Prevents known threats.

---

## Endpoint Detection and Response (EDR)

Detects and responds to advanced threats.

---

## Hardening Techniques

- Disable unnecessary services
- Apply updates
- Use antivirus solutions
