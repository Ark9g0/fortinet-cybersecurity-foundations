# Module 1: Cryptography and PKI

## Cryptography

Cryptography is the process of protecting information by converting it into a secure format.

It ensures:
- Confidentiality
- Integrity
- Authentication

---

## Types of Encryption

### Symmetric Encryption
- Same key is used for encryption and decryption
- Fast and efficient
- Problem: key distribution

### Asymmetric Encryption
- Uses public key and private key
- More secure for communication
- Used in SSL/TLS

---

## Hashing

Hashing converts data into a fixed-length output.

Key properties:
- One-way function
- Cannot be reversed
- Used in password storage

---

## Digital Signatures

Used to verify:
- Identity of sender
- Data integrity

---

## Public Key Infrastructure (PKI)

PKI is a system that manages digital certificates.

Components:
- Certificate Authority (CA)
- Digital Certificates
- Public/Private keys

---

## Real-world Use

- HTTPS websites
- Secure email communication
- VPN authentication
