# Module 6: Secure Data and Applications

## Overview

Focuses on protecting data and securing applications.

---

## Data Protection

- Encryption at rest and in transit
- Access control

---

## Data Privacy

Ensures personal data is handled securely and legally.

---

## Web Application Firewall (WAF)

Protects web applications from attacks like:
- SQL injection
- XSS

---

## Secure Email Gateway (SEG)

Filters malicious emails and phishing attempts.

---

## Content Filtering

Blocks unsafe websites and content.

---

## Application Hardening

Reducing vulnerabilities by securing application configuration.

---

## Intrusion Prevention System (IPS)

Detects and blocks malicious network activity.
