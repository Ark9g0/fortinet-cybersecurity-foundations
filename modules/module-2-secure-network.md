# Module 2: Secure Network

## Overview

Secure networking focuses on protecting data as it moves across networks and ensuring that only authorized traffic is allowed.

---

## Firewalls

Firewalls control incoming and outgoing network traffic based on security rules.

Types:
- Packet filtering firewall
- Stateful inspection firewall
- Next-generation firewall (NGFW)

---

## Secure Perimeter

Traditional model that protects internal networks from external threats.

---

## Zero Trust Security

Principle: "Never trust, always verify"

Key idea:
- Every user and device must be authenticated continuously

---

## SD-WAN

Software-defined WAN improves network performance and security by managing traffic intelligently.

---

## SASE

Secure Access Service Edge combines:
- Networking
- Security services in the cloud

---

## Network Segmentation

Dividing networks into smaller segments to limit attack spread.

---

## FortiGate

Fortinet firewall solution used for:
- Traffic filtering
- VPN access
- Threat protection

---

## Common Threats

- DDoS attacks
- Malware propagation
- Unauthorized access
